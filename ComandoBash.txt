comando git bash

1 - ls lista o que estar dentro do diretório
2 - cd mudar de diretório 
3 - pwd mostrar o caminho do diretório
4 -  ls -a mostrar as pastas ponto ponto . e ..
5 - mkdir cria uma pasta
6 - cat *nome do arquivo* mostra o que tem dentro do arquivo
7 - git init . inicializa uma pasta especial pra receber versionamento de arquivo
8 - rm -rf .git/ exclui a pasta especial .git criada diretorio
9 - git add sinalizar que é o arquivo que vai ser commitado
10 - git commit -m versiona o arquivo
11 - git config --global user.email "" assina quem estar commitando 
12 - git config --global user.name "" assina quem estar commitando
