git init . -> cria um repositorio git na pasta atual 
git status -> mostra o status do trabalho (de cada arquivo)
git add nome_arquivo -> adiciona arquivo para ser commitado
git commit -m "msg" faz commit do que estava para ser comittado

## Fluxo de trabalho ##
1. criar o diretório do projeto (pasta)
2. iniciar o git dentro do diretório (git init .)
3. edita ou criar arquivos
4 . git status
5. git add nome do arquivo
6. git commit -m "mensagem"
